O jogo

A policia de Gotham city está atras do justiceiro mascarado Batman (Para não perder o costume haha), ajude ele a fugir da policia desviando pelos obstáculos das ruas de Gotham, além disso colete os símbolos de morcegos para marcar uma maior pontuação.
Para jogar, basta acessar: Run Batman Run e utilizar o botão esquerdo do mouse para pular pelos obstáculos.
